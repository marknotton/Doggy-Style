    map-fetch

    @function map-fetch($(map), $(arglist)) { ... }
    View source

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $(map)

    $map - Usually a variable name

    		— none
    $(arglist)

    $keys - With comma delimination write the path to the disired map element

    		— none

    Returns

    Any type

    Example

    Usage

    $foo: (
  bar: (
    lorem : (
      test1: #dcfac0,
      test2: #00968b,
    ),
    ipsom: (
      test1: #f4fac7,
      test2: #c2454e,
    )
  )
);
color: map-fetch($foo, bar, lorem, test1);

    Output

    color:#dcfac0;
